# LaunchTG v4.2.1 - Анализ

## Общая информация
- **Файл**: LaunchTG.exe
- **Размер**: 120 MB
- **Тип**: PE32+ executable (64-bit Windows console)

## Защита и упаковка
- **Упаковщик**: PyInstaller
- **Шифрование**: AES (PyInstaller --key опция)
- **Статус**: Зашифрован, стандартные инструменты не работают

## Функционал приложения (из config.ini)

### Основные модули:
| Модуль | Описание |

| **Конвертеры** | Конвертация сессий (JSON, Session, Tdata) |

## Методы декомпиляции

### Неудачные попытки:
1. ❌ pyinstxtractor - "Missing cookie"
2. ❌ pyinstxtractor-ng - "Missing cookie"
3. ❌ UPX распаковка - файл не упакован UPX
4. ❌ Ручное извлечение PYZ - данные зашифрованы AES

### Рекомендуемый метод - Runtime Extraction:
```bash
# 1. Запустить скрипт мониторинга
python runtime_extract.py

# 2. Запустить LaunchTG.exe

# 3. Скрипт скопирует файлы из %TEMP%\_MEI*

# 4. Декомпилировать .pyc файлы
pip install uncompyle6
uncompyle6 -o output/ *.pyc
```

## Выводы

LaunchTG - это инструмент для массовой автоматизации Telegram:
- Спам рассылки
- Массовые репорты
- Парсинг пользователей
- Накрутка
- Инвайтинг

Приложение защищено AES шифрованием PyInstaller, что затрудняет статический анализ.
Для декомпиляции требуется runtime extraction во время выполнения программы.
